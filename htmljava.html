Tipos de Variables

  • var = "se puede cambiar"

Backticks

Es la forma mas comoda de concatenar y evitar errores de linea `${}`.

Example 1 Usando comillas

nombre = "Fernando";
frase = "me llamo " + nombre + " voy a ser ingeniero en software";

Example 2 Usando Backticks

nombre = "Fernando";
frase = `me llamo ${nombre} y voy a ser ingeniero en sofware`;

Operadores de comparacion

Comparan dos exprexiones y devuelven un valor true o false de Boloean que representa la relacion de sus valores.

  • Igual( a == b ) si solo es el mismo valor 24 o "24" true
  • Inigualdad( a != b ) no son iguales 24 o 25 true
  • Identidad( a === b ) si o si los mismos tipos de datos y el mismo valor 24 o 24 true
  • Inigualdad Estricta( a !== b ) no es el mismo tipo de dato y si el mismo valor number 24, string "24" true
  • Mayor que( a > b )
  • Mayor o igual( a >= b )
  • Menor que( a < b )
  • Menor o igual( a <= b )

Operadores Logicos

Nos devuelven un resultado a partir de que se cumpla (o no) una condicion, su resultado es booleano, y sus operandos son valores logicos o asimilables a ellos.
